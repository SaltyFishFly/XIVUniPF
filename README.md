# XIVUniPF

## 简介
XIVUniPF 是 [remote-party-finder](https://github.com/LittleNightmare/remote-party-finder) 项目的客户端实现
使用此软件可以查看FF14国服的跨大区招募板

使用 WPF + WPF UI 库实现

## 支持特性
- [x] Fluent Design 外观
- [x] 亮色/暗色主题支持
- [x] 获取跨大区招募板
- [x] 按剩余时间/任务类型/大区排序
- [x] 关键词搜索
- [ ] 后续计划实现的功能在 Issues 中查看
